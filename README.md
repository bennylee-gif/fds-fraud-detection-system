# fds-fraud-detection-system
# AI 기반 이상거래 탐지 시스템(FDS)

## 프로젝트 소개

금융 거래 데이터를 분석하여 이상 거래(Fraud)를 탐지하는 AI 기반 시스템입니다.

## 기술 스택

- Python
- Pandas
- Scikit-Learn
- XGBoost
- FastAPI
- PostgreSQL
- Streamlit

## 주요 기능

- 이상 거래 탐지
- 위험도 점수 산출
- 실시간 API 제공
- 대시보드 시각화

## 기대 효과

금융권 FDS 시스템 구조를 이해하고 머신러닝 기반 이상 탐지 모델을 구현합니다.
